# IPTproject "A simulated trading environment where users connect AI agents and test strategies using virtual capital"
Team members:
-Khalifa Tawfek

Description (stories) :
-User:
As a visitor, I want to register an account so that I can access the Simmer Market platform securily.
As a registered user, I want to add my AI agent API key so that the platform can execute simulated trades for me automatically on my behalf.
As a registered user, I want to run simulated trades using virtual money so that I can test my AI strategy without risking real funds.
As a registered user, I want to see my trading statistics profit, lost, winrate so that I can evaluate my AI strategy performance.
-Admin:
As an admin, I want to manage users and disable suspicious agents so that the platform remains fair and secure.
